# Javacreate Website
- A modern Minecraft server website built as a learning project to combine web development, AI-assisted coding, and DevOps practice.
- Includes modpack download system, server information portal, and is deployed on a VPS using Nginx for hosting.
- Focused on building real-world infrastructure for a Forge Minecraft server (Java Edition 1.20.1).

## Features
- **Modpack Download System**: Users can download the custom modpack directly from the website, with support for version management and updates.
- **Server Information Portal**: Displays server status, player count, and other relevant information to keep the community informed.
- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience across all devices, from desktops to mobile phones.
- **AI-Assisted Development**: Leveraged AI tools to assist in coding, debugging, and optimizing the website, enhancing development efficiency and learning outcomes.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: Nginx on a VPS
- **Version Control**: Git
- **AI Tools**: Various AI-assisted coding tools for development support
- **Minecraft Modding**: Forge for Minecraft Java Edition 1.20.1 and Create Mods
